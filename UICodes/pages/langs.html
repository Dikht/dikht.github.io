Programming languages

Programming language is a set of instruments for people to communicate with machines. It allows to design and develop computer programs.

HTML5 is the latest version of a popular markup language with new specific features aimed to build the structure of various webpages: forums, online stores, search systems and many more.

CSS3 describes styles of the webpages, making its visual arrangement more user-friendly. It is responsible for different fonts and blocks' colors and their appearance on the page.

Javascript is a common language for various purposes. Its possibilities may depend on the environment set, though all activity on the page connected with client and partially with server is controlled by Javascript.

Ruby on Rails is a framework based on programming language Ruby. It implements web applications and provides data exchange between database and web-server.

PHP is a widely-used scripting language for web development from the server side. Nowadays it is among the most popular languages to create a dynamic website.

NodeJS is cross-platform runtime environment combining several languages, which allows to handle queries of a client on a server side.
