Сервисы и фреймворки

Google App Engine сервис хостинга сайтов и web-приложений на серверах Google с бесплатным именеv, либо с собственным именем, задействованным с помощью служб Google.

Microsoft azure сервис, который позволяет хранить данные удаленно на сервере Microsoft. Он удобен тем, что все необходимые данные всегда доступны в любой точке мира.

Amazon aws множество платформ, которые позволяют облачно хранить ваши файлы или арендовать виртуальные сервера.

Bootstrap 3 это фреймворк, который позволяет создавать веб-сайты с помощью различных дополнений. Данный конструктор упрощает создание сайта и использует самые современные разработки.
